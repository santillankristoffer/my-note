# Notes - Private Notes & Media Vault

A privacy-first note-taking and personal vault application for Android and iOS.

Notes is designed for users who value simplicity, speed, and complete ownership of their data. Create notes, organize information, and securely store private photos, videos, documents, and other media directly on your device.

Unlike many modern productivity applications, Notes does not require an account, internet connection, cloud synchronization, or subscription plan. Your data remains under your control at all times.

## Features

### 📝 Note Taking

* Create and edit notes quickly
* Clean and minimalist interface
* Fast and responsive user experience
* Organize notes for easy access

### 🔒 Privacy Vault

* Securely store sensitive notes
* Protect private information from unauthorized access
* Keep personal content separated from regular notes

### 📸 Private Media Storage

* Store photos securely
* Save private videos
* Archive important documents
* Keep personal files protected within the application

### 📱 Fully Offline

* No internet connection required

* No external servers
* Works anywhere, anytime

### 🔐 Privacy First

* No account registration
* No subscriptions

* No unnecessary data collection
* Your data stays on your device

## Why Notes?

Many note-taking applications require:

* User accounts
* Cloud storage subscriptions
* Continuous internet connectivity
* Third-party synchronization services

Notes was built with a different philosophy:

**Your notes should belong to you.**

Everything is stored locally on your device, giving you complete control over your personal information and media.

## Installation

### Android

1. Download the latest release from Google Play.
2. Install the application.
3. Start creating notes and storing private content immediately.

No registration required.

## Privacy Policy

Notes is designed with privacy as a core principle.

* No account creation required
* No cloud storage required
* No subscription required
* No personal data sold or shared
* No mandatory internet access

All user content remains stored locally on the device unless explicitly exported by the user.
